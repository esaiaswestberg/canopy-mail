# Canopy Mail

## Development

### Requirements

- [Go 1.26.1](https://go.dev/doc/install) (or higher)
- [Wails CLI](https://github.com/wailsapp/wails/) `go install github.com/wailsapp/wails/v2/cmd/wails@latest`
- More found by running `wails doctor`.

### Getting started

First install all requirements, then we can run the project using `wails dev -tags webkit2gtk4.1` (Linux) or `wails dev` (macOS/Windows).
